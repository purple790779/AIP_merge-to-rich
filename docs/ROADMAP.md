## 2. 다음 우선순위

### 1순위: 플레이타임 확장
- `무한의농장` 벤치마킹을 기반으로 단일 보드 구조를 `지역 확장형 구조`로 전환
- `시티 뱅크 -> 골드 익스체인지 -> 젬 마켓 -> 스페이스 볼트` 순서의 월드 구조 준비
- 상점을 단순 수치 강화에서 `강화 + 해금 + 메타 선택` 구조로 재편
- `v1.5.6` 기준:
  - 현재 지역 배너 반영
  - 지역 선택 / 해금 UI 1차 반영
  - 실제 지역별 보드 분리는 다음 단계
- 보드 deadlock 구제 장치 추가

### 2순위: 경제 리밸런싱
- `v1.5.5` 기준 1차 완료
  - 수익 주기 업그레이드 체감/비용 재조정
  - 머지 보너스 기대값 상향
  - 미션 / 업적 현금 보상 비중 축소
  - 시작 레벨 업그레이드의 보드 삭제/환급 구조 제거
- 다음 단계
  - 복귀 / 오프라인 보상 수치 재조정
  - 지역 해금 비용과 메인 경제 곡선 연결
- 현금 보상 일부를 `해금 토큰 / 부스트 티켓 / 광고 배수` 같은 보상으로 치환 검토
