# 라이브 운영 로드맵

상태: Planned  
기준일: 2026-03-09

## 1. 현재 운영 상태
- 플레이 스토어 등록 및 운영 단계
- 일일 보상 구현 완료
- 복귀 보상 구현 완료
- 오프라인 보상 최소형 구현 완료
- timed reward 보류 유지 / 앱 재시작 복원 / 수동 재오픈 UX 반영 완료
- 광고 SDK / IAP는 아직 미연동

## 2. 운영 우선순위

### 1순위: 안정화
- 저장 복원 회귀 방지
- 기기별 레이아웃 QA
- 보상 중복 수령 방지
- 남아 있는 UI 문구 전수 점검
- 버전 / 문서 / 스토어 메타 동기화

### 2순위: 리텐션 고도화
- 성장 로드맵 고도화 (현재: Daily 3 / Weekly 6 / Milestone 9 구성 반영)
- 업적 메타 고도화 (현재: 7카테고리/108업적 + 레거시 랭크 + 2단 카테고리 내비 반영)
- 주간/시즌 이벤트형 미션 확장 (일일은 소수 유지)
- 도감 고도화
- 복귀 / 오프라인 보상 수치 보정

### 3순위: 수익화 도입 준비
- 보상형 광고 SDK 후보 선정
- 연결 지점 우선순위:
  1. 오프라인 보상 2배
  2. 복귀 보상 추가 지급
  3. 기간형 버프
- 정책 문서와 스토어 문구 반영 포인트 정리

### 4순위: 중장기 메타
- 환생(Prestige)
- 투자 / 타이쿤 메타
- 시즌 이벤트 운영

## 3. 다음 세션 추천 작업
- 미션 2차 확장(주간/시즌 이벤트형 조건 추가, 일일은 라이트 유지)
- 업적 밸런스 2차 조정(중후반 보상 곡선/티어 점수 튜닝)
- timed reward 수치 재조정
- 광고 SDK 연동 사전 체크리스트 정리

## 4. 선행 조건
- 수익화 도입 전:
  - 리텐션 기능 최소 3축 확보
  - 개인정보 정책 갱신 포인트 정리
  - 광고 흐름 QA
  - 스토어 문구 / 스크린샷 정리
