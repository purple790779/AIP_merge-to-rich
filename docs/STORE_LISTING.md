# 스토어 등록 문구 초안

상태: Current  
기준일: 2026-03-08

## 키워드
- 머지 게임
- 방치형 게임
- 타이쿤
- 재화 성장
- 경영 시뮬레이션
- 수집형 게임

## 짧은 설명
같은 코인을 합치고 수익을 키우는 머지 방치형 타이쿤.

## 자세한 설명
**머지 머니 타이쿤**

10원부터 시작해 지폐, 금괴, 보석, 비트코인까지 성장시키는 머지 + 방치형 타이쿤 게임입니다.

핵심 특징:
- 같은 코인을 합쳐 더 높은 단계로 성장하는 머지 플레이
- 시간이 지나면 주기적으로 수익이 쌓이는 방치형 구조
- 시작 레벨, 수익 주기, 수익 배율, 자동 병합 속도 등 업그레이드 시스템
- 일일 보상, 복귀 보상, 오프라인 보상으로 다시 들어올 이유를 만드는 리텐션 구조
- 보석 단계와 숨겨진 최종 코인까지 이어지는 성장 루프

추천 포인트:
- 짧게 켜도 진행이 보이는 템포
- 복잡한 조작 없이 반복 성장에 집중한 설계
- 수익 계산과 머지 체감이 빠르게 연결되는 구조
